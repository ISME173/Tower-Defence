# Tower Siege: The Castle Will Stand

WebGL 3D TowerDefence игра в средневеково-фентезийном сетинге. Стройте и улучшайте башни, чтобы отразить волны врагов! Постарайтесь не пропустить ни одного врага, чтобы пройти уровень на 3 звезды!

Переведена на Немецкий, Английский, Испанский, Французский, Русский, Турецкий. Поддерживает Desktop и Android платформы.
На данный момент игра проходит модерацию на Яндекс играх. Как только игра будет выложена, я добавлю ссылку на неё.

## 🛠️ Стек технологий
- Unity 6000.3.4f1
- Библиотека анимаций LitMotion (аналог DOTween)
- DI Container Reflex
- UniTask
- R3
- Addressables
- Атрибуты для инспектора NaughtyAttributes
- YG Plugin 2

## ⚙️ Как запустить проект
Проект создан на версии Unity 6000.3.4f1. [Скачать версию с официального сайта](https://unity.com/ru/releases/editor/whats-new/6000.3.4f1#notes).

### Установка и запуск
1.  Склонируйте репозиторий.
2.  Откройте проект [на версии Unity 6000.3.4f1](https://unity.com/ru/releases/editor/whats-new/6000.3.4f1#notes).
4.  Откройте сцену `MainScene` из папки `Assets/_Project/Scenes/`.
5.  Запустите сцену
